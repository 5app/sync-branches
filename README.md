# sync-branches

GitHub Action to sync one or more other branches when is updated.

## Inputs

### `githubToken`

**Required** The name of the person to greet. Default `"World"`.

## Outputs

### `time`

The time we greeted you.

## Example usage

uses: tretuna/sync-branches@v1
with:
who-to-greet: 'Mona the Octocat'
